== Description ==

Shipping method extension for the WooCommerce e-commerce plugin. This adds a free "local pickup" shipping method.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enable `Local Pickup` via the WooCommerce shipping methods options page.

== Frequently Asked Questions ==

= What are the requirements for this extension? =

Simply have WooCommerce installed and this will automatically add the shipping method. You will need to enable it from the WooCommerce options.
